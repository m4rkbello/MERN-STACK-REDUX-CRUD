MERN-STACK-REDUX 

1.CREATE A FOLDER FOR THIS PROJECT
2.INSTALL VITE
->npx create-react .
3.INSTALL NODE PACKAGES
->npm i
4. RUN THE CLIENTSIDE
->npm run dev (for vite)
5. INSTALL REDUX TOOLKIT
-> npm i @reduxjs/toolkit
5. INSTALL BOOTSTRAP
-> npm install react-bootstrap bootstrap
6. INSTALL API PROMISE-BASED HTTP LIBRARY
-> npm i axios
7. INSTALL MONGODB  
-> mongodb compass 
8. INSTALLING THE NEEDED FOR SERVERSIDE 
-> npm i cors (cross origin resource Sharing)
-> npm i express (emanage ang servers ug routes)
-> npm i i mongoose
-> npm i nodemon
9. INSTALL REDUX-THUNK
-> npm install redux-thunk
